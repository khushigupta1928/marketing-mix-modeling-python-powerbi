📈 Marketing Mix Modeling using Python & Power BI
📌 Project Overview

This project demonstrates an end-to-end Marketing Mix Modeling (MMM) solution using Python, Machine Learning, and Power BI. The objective is to analyze the impact of different marketing channels on sales, measure campaign performance, and provide data-driven business recommendations through interactive dashboards.

🎯 Project Objectives
Analyze the effectiveness of TV, Radio, Social Media, and Influencer Marketing.
Identify the marketing channels contributing the most to sales.
Predict Sales using a Linear Regression model.
Calculate Marketing Cost, Profit, and ROI.
Build interactive Power BI dashboards for business decision-making.

💡 Key Insights
TV advertising contributed the highest sales.
Marketing campaigns achieved an average ROI of approximately 154.73%.
Higher marketing investment generally resulted in higher sales.
Mega influencers delivered the strongest campaign performance.
The Linear Regression model achieved excellent prediction accuracy with an R² Score of 99.6%.
